# frontend

Vue 3 + TypeScript + Vite admin app for the Role Inference Service. See the
"Frontend (Vue 3 + TypeScript)" section in the repo root [README](../README.md)
for architecture, and "Setup" there for how this fits into the full stack.

```sh
npm install
npm run dev      # Vite dev server on :5173, proxies /api to :8000
npm run build    # type-checks (vue-tsc) then outputs static assets to dist/
```
